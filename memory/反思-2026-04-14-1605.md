# 反思-2026-04-14-1605：批量图片生成任务

## 任务概述
用户要求生成 OpenClaw 部署服务广告图20张，每张有详细的文案和画面描述。

## 过程
- 尝试多种批量方法：TCP socket、PowerShell循环、exec后台进程
- exec进程被SIGKILL，无法后台持久运行
- 最终用image_generate工具(count=4)分5批生成，每批4张
- MiniMax API key格式问题：sk-cp-xxx格式返回invalid key，但OpenClaw内置image_generate工具可用

## 教训
- Windows exec后台进程容易被kill，不适合长时批量任务
- OpenClaw的image_generate工具内部已正确路由到MiniMax API
- 批量生成最佳方式：直接调用image_generate(count=4)分批跑
